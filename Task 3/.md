# 🛡️ Nexus IDS - Network Intrusion Detection System (Task 3)

Welcome to **Nexus IDS**, an interactive Network Intrusion Detection System and Attack Simulation Tool. This project demonstrates how to monitor network traffic in real-time and detect suspicious activities like port scanning and Denial of Service (DoS) attacks.

## 🌟 Features

- **Real-time Monitoring:** Continuously monitors network traffic using `scapy` and `psutil`.
- **Attack Detection:** Identifies malicious behavior patterns such as Rapid Port Scans and High Volume Traffic (DoS).
- **Interactive Dashboard:** Live updates on network events streamed to the frontend via WebSockets.
- **Attack Simulation Tool:** Includes a built-in Python script (`test_attack.py`) to safely simulate attacks against your local environment and verify the IDS is working.
- **Full-Stack Architecture:** Built with a FastAPI backend and a responsive frontend interface.

## 🛠️ Technologies Used

- **Backend Framework:** FastAPI, Uvicorn
- **Networking/Security:** Scapy, WebSockets, Sockets
- **System Monitoring:** Psutil
- **Frontend:** HTML, CSS, JavaScript (Dashboard)

## 🚀 How to Run Locally

### 1. Setup the Backend
Clone the repository and install the dependencies:
```bash
git clone https://github.com/VP5412/codealpha_tasks.git
cd codealpha_tasks/"Task 3"
pip install -r requirements.txt
```

Start the IDS backend server (make sure you have appropriate permissions/sudo for packet sniffing if required):
```bash
cd backend
# Command to run the FastAPI server (e.g., uvicorn main:app --reload)
```

### 2. Open the Frontend Dashboard
Navigate to the `frontend` directory and open the application in your browser to view the real-time Nexus Dashboard.

### 3. Run Attack Simulations
Open a separate terminal to run the attack simulation tool and watch the dashboard react!

**Simulate a Port Scan & DoS Attack:**
```bash
cd "Task 3"
python test_attack.py 127.0.0.1 --attack both
```

**Available Attack Options:**
- `--attack scan`: Simulates a rapid port scan.
- `--attack dos`: Simulates high volume UDP traffic.
- `--attack both`: Runs both simulations consecutively.
